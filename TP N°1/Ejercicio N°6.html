Introducción de HTML

Este es un curso que pretende introducir al lector en el mundo del
HTML. No se requiere conocimientos previos, se empezará desde cero.
No hay que saber programar para utilizar HTML, basta
aprender un par de cosas sencillas para empezar a crear páginas.

Es Muy FACIL aprender HTML. Tan solo necesitas un editor de texto, un
curso de HTML y un navegador Web (como Netscape o internet Explorer),
ni siquiera necesitas estar conectado a Internet
